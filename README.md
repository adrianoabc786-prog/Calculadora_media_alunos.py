# Calculadora_media_alunos.py
# 🎓 Sistema de Média Acadêmica em (Python)

Este é um projeto simples desenvolvido durante o primeiro semestre do curso de **Engenharia de Software**. O objetivo é aplicar conceitos fundamentais de lógica de programação utilizando a linguagem Python.

## 📝 Descrição
O sistema solicita o nome de um aluno e duas notas bimestrais. Ele calcula a média aritmética e informa se o aluno foi **Aprovado**, está em **Recuperação** ou foi **Reprovado**, baseando-se em regras de decisão (`if/elif/else`).

## 🚀 Tecnologias e Conceitos Utilizados
* **Python 3**: Linguagem principal.
* **Casting**: Conversão de tipos de dados (`float`).
* **F-Strings**: Formatação de strings para saída de dados organizada.
* **Estruturas Condicionais**: Lógica para determinação do status do aluno.
